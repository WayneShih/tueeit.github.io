# 勞動黨新竹縣議員高偉凱提案加強勞檢　並呼籲籌組工會

張貼者：2011年4月14日 下午10:36 Ming Che Lin  [ 已更新 2011年4月14日 下午10:43 ]

![20110414_stand_by_road](/images/20110414_stand_by_road.jpg)

4月1日勞動黨新竹縣議員高偉凱在縣議會臨時會中提案，指過勞死及工時過長引發相關疾病，已成為嚴重工安問題。就許多縣民的工作所在地，提請縣政府協同新竹市政府、科管局加強勞動檢查，徹查新竹縣市及科學園區事業單位有無違法採用責任制，或超時加班、未給付加班費情事。

高偉凱所屬的勞動黨桃竹苗勞工服務中心同時並在新竹地區租用大型看板及「站路口」（這是勞動黨選舉時最引人注目的方式之一，在上班時段於工業區或科學園區週遭，以及假日的大賣場、夜市入口，舉手舉牌），呼籲新竹縣市勞工重視自身權益。就算要求官方加強檢查，難免會有疏漏；勞工若認為公司有違法責任制、超時加班之虞，歡迎跟服務處聯絡詢問。另外，法律的規定是一回事，現實狀況有很多明明違法卻查不出來，或者查出來也硬說成合法的情形。只有靠勞動現場的工人組織起來，一起反對這個不合理的狀況，才可能改變。

高偉凱服務處正在協助組織電子電機資訊產業工會，歡迎相關產業勞工報名加入，共同來改變不合理的責任制。

# 官商勾結？勞檢放水！台灣電子電機資訊產業工會赴科管局抗議

張貼者：2011年10月18日 下午4:09 Ming Che Lin  [ 已更新 2011年10月18日 下午4:10 ]

時間：中華民國100年10月19日上午10:00 

地點：科學工業園區管理局（竹科新安路1號） 

連絡人：高偉凱（電資工會顧問、勞動黨新竹縣議員0910-170417）  

說明：針對U公司某部門強迫員工加班、否則扣薪的問題，電資工會向科管局進行檢舉；科管局檢查結果認無違法。電資工會認為科管局重大疏失： 

電資工會於今年8月11日發函檢舉，並經科管局要求提供申訴員工姓名，並於8月22日發函補充請求檢查該部門「內部宣導」文件及班費收支記錄。但科管局遲至9月20日才針對檢舉結果回函，違反勞動檢查14日內（書面）回覆的規定。

勞動檢查法施行細則規定檢查單位勞檢時應通知工會陪同；但科管局在勞委會進一步認定前，即先行排除電資工會作為勞檢法中的「工會」，逕行認定該工會僅限廠場／企業工會，罔顧法無明訂、且U公司無工會、但本會在U公司有會員且原申訴人亦為本會會員的事實。

科管局既已要求電資工會提供申訴員工姓名，但檢查時卻僅採用公司說法，不詢問勞方意見。

科管局不檢查本會提出之相關文件，僅檢查員工薪資帳冊；如該筆扣款未記錄於薪資明細中，自然查不到。

科管局作為科學園區之主管機關，不屬任何地方政府管轄，也沒有民意機關直接監督，影響科技業員工權益鉅大；因此電資工會決定到科管局抗議，並將向監察院陳情請求調查。

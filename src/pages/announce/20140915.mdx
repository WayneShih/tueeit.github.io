# 抗議商業週刊「Line時代　溝通必勝手冊」一文

張貼者：2014年9月15日 上午11:42 Ming Che Lin  [ 已更新 2014年9月15日 上午11:42 ]

商業週刊第1399期刊登了一篇「[Line時代　溝通必勝手冊](http://www.businessweekly.com.tw/KIndepArticle.aspx?ID=23522)」文章，內文提到如「54.7％的主管，希望Line訊息在半小時內就得到回應」、「有7成百大企業的主管，下班後會Line員工」等，暗示下班後on call的狀況是正常合理的。我們認為不應該鼓勵這樣的風氣，於是寄了以下內容向商業週刊抗議。  

致貴刊編輯部:  

拜讀貴刊本期「Line時代　溝通必勝手冊」一文，對於內文錯誤之觀念，本會感到遺憾。文章中提及使用Line的一些職場倫理，但貴刊一直在迴避一個事實：在出勤時間外，以APP通知公事，並要求即時回覆，即構成強制加班  

勞工的下班時間本應是用來休息、與家人共享天倫之樂的，除非有緊急情事，否則不應打擾。就算有事，也應遵守勞基法之規定，這是為了保障勞工健康與生活品質。然而貴刊文章所倡導的Line溝通術，企圖把這個行為正當化，勞工等於24小時on call，無法休息。  

依法若未經勞資會議同意，強制要求加班，恐違反勞基法第30條、第32條、第42條等多項法令。即使取得勞資會議同意，亦須遵守第32條所規定之上限，並給予加班費。更何況Line本身有對話記錄之功能，可作為證據，任何違法行為將無所遁形。  

在其他國家如法國已立法規定，企業不得於晚上6 點到早上9 點的非正常工作時間，發送郵件或打電話給員工；德國部份企業已跟進這項措施，德國政府亦已開始研究立法規範的可性行。  

今年6月，我國已有濫用通訊軟體導致過勞死的案例。貴刊身為國內重要的財經雜誌，在探討Line相關職場倫理之餘，亦應提及相關法律條文、權利義務關係、on call對生理心理之影響及企業之社會責任，提供讀者正確觀念，以建立健全的勞資關係。同時，亦請告誡貴刊讀者內的主管群，切勿以身試法，濫用通訊APP，害人害己。

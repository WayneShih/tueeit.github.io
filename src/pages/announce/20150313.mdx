# 伙食費免稅額調整了，但你加薪了嗎？

張貼者：2015年3月13日 上午9:04 Chuck Lee  [ 已更新 2015年3月13日 下午5:18 ]

財政部修改相關法規\[1][2]，將伙食費的免稅額從1,800元調整為2,400元，希望藉此鼓勵企業加薪。  

但我們不禁要問，這有效嗎？  

電資工會最近才處理一件勞資爭議案，勞方到某科技公司上班，**約定薪資是27,000元**，實際領到薪資單上面寫本薪25,200元、伙食津貼1,800元。

而公司在幫勞方**投保時申報的薪資是25,200元**，勞方詢問我們後才知道伙食津貼也算工資，公司其實應該以27,600元幫她投保（也就是月薪27,000對應的投保級距）！

公司用 25,200元投保是違法的，勞方不只在每個月勞工**退休金提撥部份有損失**，之後任何向勞保局請領的給付（例如**生育給付**、**傷病給付**）也都會短少。

所以財政部這項措施，要幫助到勞工恐怕不容易。 

以前面案例的勞工來說，財政部的期望是本薪25,200元，伙食津貼增加為2,400，薪資增為27,600（下表狀況一）；

但身為第一線的勞工，我們認為薪資單變成本薪24,600、伙食津貼2,400，薪資維持27,000的機率比較高（下表狀況二）。

對勞工來說，薪水不只不會增加，惡質一點的甚至公司還可能會把勞保級距調低（雖然這是違法的）；唯一的好處是每年五月申報所得稅時可以少繳一點點錢──但薪水低的勞工，本來要繳的稅就不多，省那點零頭實在高興不起來。  

|           |     調整前 | 調整後（狀況一） | 調整後（狀況二） |
| --------- | ---------: | ---------------: | ---------------: |
| 本薪      |      25200 |            25200 |            24600 |
| 伙食津貼  |       1800 |             2400 |             2400 |
| **薪資 ** | **27000 ** |        **27600** |        **27000** |

總之本會要提醒並呼籲大家： 

1. **公司在投保時若把伙食費扣除是違法的**（另外全勤獎金、職務加給等等也不能扣除，細說下去就要談到「工資」的定義，可以參考文章 \[2][3]）。

2. 財政部這項措施，能幫到勞工的不多，幫企業節稅的成份比較大。

話說回來，我們認為要期盼政府修什麼加薪四五六七法讓企業加薪，本來就是不切實際的想法。 

最有效的加薪方式，還是要靠**勞工團結起來集體與企業協商**。 

最後要說的那句話，我們已經講過太多次了，不信的話大家問問專業的五樓。（五樓：請大家成立或加入工會）

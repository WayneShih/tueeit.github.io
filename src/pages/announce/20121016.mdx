# 勞工陳情 主委落跑 工會意見未獲回應

張貼者：2012年10月17日 上午12:30 Ming Che Lin  [ 已更新 2012年10月17日 上午12:30 ]

台灣電子電機資訊產業工會與反M型社會聯盟，13日上午前往文化大學，要向到文化大學上課的潘世偉陳情，反對潘日前提出將依職類別訂定不同工時規範的說法。陳情成員十餘人在校門外集結，手持標語和陳情書，希望親自把意見傳達給潘世偉。  

電資工會秘書長林名哲指出，法定工時是限制雇主可支配勞工的時間上限，並非勞工實際進行產出的時間，潘世偉卻以實際工作時間難計算為藉口，要鬆綁法律規範。林名哲認為，勞基法的工時上限考量勞工健康與生活需求，是不可侵犯的人權，不因職業別而有差異。參與陳情的郭姓工程師表示，現行的勞基法第八十四條之一，已經間接導致科技業的病態責任制，若進一步放寬，將使工程師的工作時間無限延長。  

陳情活動原本預計在校內進行，但因文大校方反彈，協調之下改到校門口，電資工會請警方轉達要求潘世偉出面。不料活動剛開始，工會接獲消息指出，原本該從十點上課到中午的潘世偉，早在十點前就由警方護送離去。一行人將陳情書交由文大校方代表轉達。勞工感嘆，千里迢迢來只為與主委當面討論，怎料見主委這麼難。

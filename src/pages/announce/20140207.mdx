# 和PTT網友討論罷工問題

張貼者：2014年2月7日 下午5:10 Ming Che Lin  [ 已更新 2014年2月7日 下午5:11 ]

幾天前有網友在PTT上回應「為何台灣人不罷工？」一文，有些地方是需要修正的，本會特此對其中某些觀點提出說明。  

罷工在台灣很少發生，通常是勞工已經到沒有其他後路，才會使用這種手段（例如2011年太子汽車罷工，是已經被欠薪水九個月；2012年華隆罷工，也是工廠已經換招牌，本來的公司沒在生產）。為什麼罷工在台灣這麼少發生，法律過多的限制確實是部份原因，相關法律也還有不合理的地方需要修改；但其實罷工更大的困難是源自勞方不團結！工會不是要勞工一不爽就罷工，遇到公司的刁難或不合理措施可先依法律程序尋求救濟。但因為勞工往往勢單力薄，才要透過組織工會，匯集眾人的力量向企業或政府施壓。如果多數勞工連加入工會都不敢了，談什麼罷工呢？
